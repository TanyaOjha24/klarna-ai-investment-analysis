# klarna-ai-investment-analysis
# Deconstructing Klarna's $40M AI Claim: A Business Value Attribution Analysis

## Project Summary
An analytical case study evaluating Klarna's AI customer support investment using public financial filings, 
operational metrics, and original attribution analysis.

## The Question
Klarna publicly claimed $40M in profit improvement from AI. 
This project investigates how much of that is genuinely attributable to AI 
versus parallel restructuring initiatives.

## Key Finding
AI's isolated year-one contribution is estimated at $15-25M, not $40M. 
The larger strategic value was capacity unlock -- enabling 20% volume growth 
without proportional headcount increases.

## What's Inside
- Financial model: 5-year NPV and IRR analysis
- Monte Carlo simulation: 1,000 scenarios with sensitivity analysis
- Value driver decomposition: Labor efficiency, repeat inquiry reduction, capacity unlock
- Counterfactual analysis: Three attribution scenarios
- Operational KPI layer: Cost per ticket, FCR, resolution time, automation rate
- Reality check: Where the model was right and wrong vs actual outcomes

## Data Sources
- Klarna F-1 SEC Filing (2025)
- Klarna Annual Reports 2022-2023
- Klarna Press Release Feb 27 2024
- McKinsey Generative AI Report 2023
- CX Dive / Klarna Q1 2025 Earnings
